# Service Abstraction Layer

## Dependencies

- `python3.5` Python Interpreter
- `python3-rdflib` [RDFLib](https://github.com/RDFLib/rdflib) python module
- `make` Make build system

## Building Ontology

```shell
make var/sal2instances.ttl
```

## Building Model

```shell
make var/defaultbuilding.ttl
```

## Executing Tests

```shell
make test
```

## Doing Everything

```shell
make
```
